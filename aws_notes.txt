
UPLOAD STUFF TO GITHUB
1. git init
2. git remote add origin (github url)
3. git add- A (all of the stuff)  --- from your working directory to staging area
4. git commit -a                  --- from staging area to local repo
5. git push -u origin master      --- from local repot to remote repo

For changing Stuff on Github from local:
1. cd to dir first
2. git status
3. git add -A
4. git commit -m 'message'
3. git pull
4. git push

For changing Stuff on AWS EC2 from updated Github, do the exact same things:
1. cd to dir first
2. git add -A
3. git pull
